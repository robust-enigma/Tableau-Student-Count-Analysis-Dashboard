## Student-Count-Analysis (Interactive Dashboard creation using Tableau)

## Key Highlights

- Connected **Tableau Desktop** to **SQL Server** database to analyze student mental health and depression patterns.  
- Extracted and visualized data containing fields such as **Gender, Age, Academic Pressure, Study Satisfaction, Sleep Duration, Dietary Habits, Study Hours, Financial Stress,** and **Depression Status**.  
- Created **interactive dashboards** to explore relationships between academic, lifestyle, and psychological factors.  
- Designed visual summaries for **Sleep Duration, Study Hours, Financial Stress, Academic Pressure,** and **Study Satisfaction** to identify key behavioral trends.  
- Implemented **data blending** and **calculated fields** to derive correlations and student counts across multiple parameters.  
- Used various charts (**bar, pie, line, shape**) and **color encoding** for clear and intuitive storytelling.  
- Applied **SQL queries** for data extraction, filtering, and preprocessing before importing into Tableau.  
- Delivered insights to highlight potential contributors to **student depression** and patterns across demographics.  
- Ensured optimized performance by creating **extracts (.hyper)** for smooth dashboard interaction and portability.
